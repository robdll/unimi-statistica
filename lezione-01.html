1. Introduzione

Python è un linguaggio interpretato, dinamicamente tipizzato e con una sintassi minimale: per cominciare a usarlo basta assegnare un valore a una variabile. In questa lezione useremo come "scuola" un dataset (semplificato) di supereroi: nome, identità, anno di prima apparizione, peso, altezza, ecc.

Come usare la pagina. Premi ▶ Run sui blocchi di codice per eseguirli direttamente nel browser. Il primo run scarica Pyodide (l'interprete Python in WebAssembly) – richiede qualche secondo, le esecuzioni successive sono istantanee.

2. Tipi di dati semplici

I tre tipi fondamentali di Python sono bool (con i valori True e False), int (interi) e float (numeri in virgola mobile). Non esiste un tipo carattere: le stringhe sono direttamente oggetti del linguaggio.

Per scoprire il tipo di un valore puoi usare la funzione predefinita type(): 

first_appearance = 1971
weight = 71.6
is_hero = True

print(type(first_appearance))
print(type(weight))
print(type(is_hero))

Per i numeri molto grandi o molto piccoli si usa la notazione scientifica: 1E9 vale un miliardo, 1E-9 un miliardesimo. Prova: 

a = 1E9
b = 1E-9
print(a, b)
print(type(a), type(b))
Tipizzazione dinamica ma forte. Il tipo di una variabile può cambiare riassegnandola, ma non è "tutto e niente": le operazioni vengono comunque controllate a runtime e Python lancia un errore se hanno senso solo per certi tipi (es. "abc" + 1).

Cosa stampa il seguente codice?

print(type(3.0))

  • <class 'int'>
  • <class 'float'>
  • <class 'number'>
  • Errore di sintassi
La presenza del punto rende 3.0 un letterale di tipo float, anche se rappresenta un valore matematicamente intero.

Python supporta tutti i tipici operatori binari, scritti in notazione infissa (l'operatore in mezzo ai due operandi):

Attenzione alle due divisioni! / restituisce sempre un float, anche quando il risultato è "esatto" (4 / 2 vale 2.0, non 2). Per ottenere un intero usa //.

Gioca con questa "calcolatrice" – cambia i valori e i simboli a piacere:

a = 17
b = 5

print("a - b  =", a - b)

print("a % b  =", a % b)    # resto

print("a == b ?", a == b)

Dato un certo numero di giorni, scrivi un'espressione che stampi quante settimane intere contiene e quanti giorni avanzano. Per esempio, 23 giorni = 3 settimane e 2 giorni.

4. Le liste

Una lista è una struttura dati eterogenea (può contenere oggetti di tipo diverso) e ad accesso posizionale: si dichiara con parentesi quadre, separando gli elementi con virgole. 

iron_man = ['Iron Man',
            'Tony Stark',
            'Long Island, New York',
            'Marvel Comics',
            198.51,
            191.85,
            'M',
            1963,
            'Blue',
            'Black',
            85,
            'high']

print(type(iron_man))
print("Numero di campi:", len(iron_man))

4.1 Accesso e slicing

Si accede agli elementi indicandone la posizione tra parentesi quadre. La numerazione parte da 0. Indici negativi contano dalla fine: -1 è l'ultimo elemento, -2 il penultimo, e così via. 

iron_man = ['Iron Man', 'Tony Stark', 'Long Island, New York',
            'Marvel Comics', 198.51, 191.85, 'M', 1963,
            'Blue', 'Black', 85, 'high']

print(iron_man[0])    # primo elemento
print(iron_man[1])    # identità
print(iron_man[-1])   # ultimo
print(iron_man[-2])   # penultimo

Lo slicing permette di estrarre una sottosequenza con la sintassi lista[inizio:fine]. Il primo indice è incluso, il secondo è escluso: 

iron_man = ['Iron Man', 'Tony Stark', 'Long Island, New York',
            'Marvel Comics', 198.51, 191.85, 'M', 1963,
            'Blue', 'Black', 85, 'high']

print(iron_man[4:6])     # altezza e peso
print(iron_man[-4:-2])   # gli occhi e i capelli
print(iron_man[:3])      # primi tre
print(iron_man[-3:])     # ultimi tre
Perché "fine esclusa"? Permette scritture comode come l[i:i+n] per estrarre n elementi a partire da i, e di concatenare due slice senza sovrapposizioni: l[:k] + l[k:] == l.

Data l = [10, 20, 30, 40, 50], quanto vale l[1:4]?

  • [10, 20, 30, 40]
  • [20, 30, 40]
  • [20, 30, 40, 50]
  • [20, 30]
L'indice 1 è incluso (valore 20), l'indice 4 è escluso (quindi non si arriva al 50).

4.2 Operatori, funzioni e metodi

Sulle liste possiamo usare tre tipi di "strumenti" che è bene distinguere:

  • gli operatori: simboli posti tra/prima degli operandi (es. in, del);
  • le funzioni: si invocano col nome seguito da parentesi (es. len(l));
  • i metodi: appartengono all'oggetto e si invocano con la dot notation (es. l.sort()).
names = ['Aquaman', 'Ant-Man', 'Batman', 'Black Widow',
         'Captain America', 'Daredavil', 'Elektra', 'Flash',
         'Green Arrow', 'Human Torch', 'Hancock', 'Iron Man',
         'Mystique', 'Professor X', 'Rogue', 'Superman',
         'Spider-Man', 'Thor', 'Northstar']

# Operatore "in"
print('Hancock' in names)
print('Wolverine' in names)

# Funzione len()
print("Quanti supereroi:", len(names))

# Metodo .sort() — modifica IN PLACE e non restituisce nulla
names.sort()
print(names[:5])
In place vs. nuova copia. Il metodo sort() modifica la lista in place e restituisce None. Se vuoi una nuova lista ordinata senza toccare l'originale, usa la funzione sorted(l).

L'operatore del elimina un elemento data la sua posizione:

names = ['Aquaman', 'Ant-Man', 'Batman']
print("Prima:", names)

del names[0]
print("Dopo:", names)
print("Lunghezza:", len(names))

4.3 Lambda e argomenti opzionali

Molti metodi e funzioni accettano argomenti opzionali, identificati per nome e con un valore di default. Per esempio sort(reverse=True) inverte il verso dell'ordinamento.

Una lambda è una funzione anonima, comoda per parametrizzare ordinamenti e trasformazioni: lambda x: <espressione>. 

successore = lambda n: n + 1
print(successore(9))

names = ['Aquaman', 'Ant-Man', 'Batman', 'Captain America', 'Thor']

# Ordina per lunghezza del nome (crescente)
names.sort(key=lambda n: len(n))
print(names)

# Ordina alfabeticamente al contrario
names.sort(reverse=True)
print(names)

Esercizio 2 — Ordinare per ultima lettera

Data la lista di nomi qui sotto, ordinala in modo che vengano mostrati in base all'ultima lettera del nome (in ordine alfabetico).

Suggerimento: come key serve una funzione che, dato un nome, restituisca il suo ultimo carattere (l'indice -1 di una stringa). 

names = ['Aquaman', 'Batman', 'Iron Man', 'Thor', 'Hulk', 'Flash']

# Tua soluzione qui sotto:
# names.sort(key=...)
# print(names)

names = ['Aquaman', 'Batman', 'Iron Man', 'Thor', 'Hulk', 'Flash']
names.sort(key=lambda n: n[-1])
print(names)

Cosa stampa il seguente codice?

l = [3, 1, 2]
print(l.sort())
  • [1, 2, 3]
  • None
  • [3, 1, 2]
  • Errore
list.sort() ordina la lista in place e restituisce None. Per stampare la lista ordinata serve una riga separata (oppure usare sorted(l)).

5. Le tuple

Una tupla è una lista immutabile: una volta creata, i suoi elementi non si possono modificare. Si dichiara con parentesi tonde. 

rogue = ('Rogue', 'Anna Marie', 'Caldecott County, Mississippi',
         'Marvel Comics', 173.1, 54.39, 'F', 1981,
         'Green', 'Brown / White', 10, 'good')

print(rogue[1])
print(len(rogue))

# Tentare di modificare una tupla lancia un'eccezione
try:
    rogue[-2] = 70
except TypeError as e:
    print("Errore:", e)
Tupla con un solo elemento. (1) è interpretato come il numero 1 tra parentesi. Per indicare una tupla con un unico elemento serve la virgola: (1,).

Quando preferire una tupla a una lista? Quando vuoi proteggere i dati da modifiche accidentali (è una garanzia, oltre che un suggerimento per chi legge il codice) e in molti casi è anche un po' più efficiente.

Quale espressione crea una tupla con un solo elemento?

  • (42)
  • (42,)
  • [42]
  • tuple(42)
La virgola finale è ciò che distingue una tupla da un'espressione tra parentesi. tuple(42) non funziona perché si aspetta un iterabile.

6. Le stringhe

Le stringhe si comportano come tuple di caratteri: si possono indicizzare e "slicare", ma non si possono modificare in place. 

name = 'Anna Marie'
print(name[3])         # un singolo carattere
print(name[:4])        # primi 4 caratteri
print(name[-5:])       # ultimi 5
print(len(name))

# Le stringhe sono immutabili
try:
    name[0] = 'a'
except TypeError as e:
    print("Errore:", e)

Per "modificare" una stringa, in realtà se ne crea una nuova: 

name = 'Anna Marie'
name = name.lower()
print(name)
print(name.replace('anna', 'Joanne'))
print(name.upper().split(' '))

7. Gli insiemi (cenni)

Un set rappresenta una collezione finita di elementi distinti e non ordinati. Lo useremo poco nel corso, ma è un tipo di dato molto utile per controllare unicità o per fare operazioni "alla matematica" (unione, intersezione…). 

a = {1, 2, 3, 3, 2, 1}     # i duplicati spariscono
b = {3, 4, 5}

print("a =", a)
print("a | b =", a | b)    # unione
print("a & b =", a & b)    # intersezione
print("a - b =", a - b)    # differenza
print(2 in a)

8. I dizionari

Un dizionario associa chiavi a valori (in modo analogo al concetto matematico di funzione). È spesso il modo più leggibile per rappresentare un "record". 

rogue = {'name': 'Rogue',
         'identity': 'Anna Marie',
         'birth_place': 'Caldecott County, Mississippi',
         'publisher': 'Marvel Comics',
         'height': 173.1,
         'weight': 54.39,
         'gender': 'F',
         'first_appearance': 1981,
         'eye_color': 'Green',
         'hair_color': 'Brown / White',
         'strength': 10,
         'intelligence': 'good'}

print(rogue['identity'])
print('publisher' in rogue)

# Accesso in scrittura: aggiunge la coppia se la chiave non esiste
rogue['affiliation'] = 'X-Men'
print(rogue['affiliation'])

# Accesso a una chiave inesistente solleva KeyError
try:
    print(rogue['power'])
except KeyError as e:
    print("Chiave assente:", e)
Quando usarli? Se i campi sono concettualmente nomi e non posizioni (come per i record), i dizionari rendono il codice molto più leggibile: rogue['identity'] è chiarissimo, mentre rogue[1] obbliga a ricordare l'ordine.

Quale di queste affermazioni sui dizionari è falsa?

  • L'operatore in verifica la presenza di una chiave.
  • Assegnare a una chiave inesistente la aggiunge al dizionario.
  • Si può accedere ai valori per posizione, come nelle liste.
  • L'accesso in lettura a una chiave inesistente solleva KeyError.
I dizionari sono associativi: l'accesso avviene per chiave, non per posizione. Le altre tre affermazioni sono vere.

Esercizio 3 — IMC del supereroe

Dato il dizionario iron_man qui sotto, calcola il suo indice di massa corporea (BMI = peso in kg / altezza in m al quadrato) e stampalo arrotondato a una cifra decimale.

Suggerimento: l'altezza è in centimetri, quindi convertila in metri (dividendo per 100). Per l'arrotondamento c'è la funzione round(x, 1). 

iron_man = {'name': 'Iron Man', 'height': 198.51, 'weight': 191.85}

# Tua soluzione qui sotto
# bmi = ...
# print(...)

iron_man = {'name': 'Iron Man', 'height': 198.51, 'weight': 191.85}

altezza_m = iron_man['height'] / 100
bmi = iron_man['weight'] / (altezza_m ** 2)
print(f"BMI di {iron_man['name']}: {round(bmi, 1)}")

9. Strutture di controllo

Python organizza i blocchi di codice tramite l'indentazione: non servono parentesi graffe né begin/end, ma è obbligatorio mantenere coerente la spaziatura.

9.1 Selezione: if/elif/else

first_appearance = 1981

if first_appearance < 1960:
    epoca = "età d'oro"
elif first_appearance < 1980:
    epoca = "età d'argento"
else:
    epoca = "età moderna"

print(f"{first_appearance}: {epoca}")

9.2 Iterazione: for

Il for di Python non è un ciclo numerato classico: itera sugli elementi di un oggetto iterabile (lista, tupla, stringa, dizionario, range…). 

names = ['Aquaman', 'Batman', 'Thor']
for n in names:
    print(n, '→', len(n), 'caratteri')

Se ti serve un indice usa enumerate; se vuoi un ciclo "alla C" usa range: 

names = ['Aquaman', 'Batman', 'Thor']

for i, n in enumerate(names):
    print(i, n)

print("---")

for i in range(3):
    print("iterazione", i)

9.3 Esempio: frequenze assolute

Mettiamo insieme dizionari, for e if per contare quanti supereroi sono apparsi per la prima volta in ogni anno (assegnando None a quelli con anno mancante): 

years = [1941, 1962, None, None, 1941,
         1964, None, 1940, 1941, 1961,
         None, 1963, None, 1963, 1981,
         None, None, 1962, 1979]

counts = {}
for y in years:
    if y in counts:
        counts[y] += 1
    else:
        counts[y] = 1

print(counts)
Più elegante con defaultdict. Vedremo nella sezione "Moduli" come riscrivere questo conteggio senza il controllo if.

Quale di queste affermazioni sul for di Python è vera?

  • È un ciclo numerato: la variabile vale 0, 1, 2, … fino a un limite.
  • Itera sugli elementi di qualsiasi oggetto iterabile.
  • Funziona solo con le liste.
  • Richiede sempre la funzione range.
Il for di Python è un foreach: scorre gli elementi di qualsiasi iterabile. Per avere un ciclo numerato basta iterare su range(n).

10. Funzioni

Le funzioni si definiscono con la parola chiave def. Il corpo è indentato, e il valore restituito è specificato con return (in mancanza di return, la funzione restituisce None). 

def get_sorted_counts(sequence):
    """Restituisce le frequenze assolute degli elementi, ordinate dal più al meno frequente."""
    counts = {}
    for x in sequence:
        if x in counts:
            counts[x] += 1
        else:
            counts[x] = 1
    pairs = list(counts.items())
    return sorted(pairs, key=lambda p: p[1], reverse=True)


years = [1941, 1962, None, None, 1941, 1964, None, 1940, 1941, 1961,
         None, 1963, None, 1963, 1981, None, None, 1962, 1979]

print(get_sorted_counts(years))
Docstring. Le triple virgolette subito sotto la definizione sono la docstring: è la documentazione "ufficiale" della funzione, accessibile via help() o l'attributo .__doc__.

11. Importare moduli

Un modulo è un file (o pacchetto) che contiene definizioni di funzioni, classi e variabili. Si importa con import; si può importare tutto il modulo, solo alcuni suoi elementi, o assegnargli un nome alternativo. 

# Forma 1: importa l'intero modulo, accesso via "math.NOME"
import math
print(math.pi)
print(math.sqrt(2))

# Forma 2: importa solo alcuni nomi
from math import floor, ceil
print(floor(3.7), ceil(3.2))

# Forma 3: alias (convenzione molto diffusa)
import statistics as stats
print(stats.mean([1, 2, 3, 4]))

Riscriviamo il conteggio delle frequenze sfruttando defaultdict dal modulo standard collections: niente più if! 

from collections import defaultdict

def get_sorted_counts(sequence):
    counts = defaultdict(int)   # chiavi mancanti → 0
    for x in sequence:
        counts[x] += 1
    return sorted(counts.items(), key=lambda p: p[1], reverse=True)

years = [1941, 1962, None, None, 1941, 1964, None, 1940, 1941, 1961,
         None, 1963, None, 1963, 1981, None, None, 1962, 1979]

print(get_sorted_counts(years))
Convenzioni. Quando lavoreremo con dati useremo molto numpy e pandas. La convenzione (rispettala!) è importarli così: import numpy as np e import pandas as pd.

Cosa fa defaultdict(int)?

  • Crea un dizionario in cui tutte le chiavi devono essere interi.
  • Crea un dizionario in cui le chiavi mai assegnate restituiscono 0.
  • Crea un dizionario di numeri interi crescenti.
  • È un alias di dict.
defaultdict(int) imposta come fabbrica per i valori assenti la chiamata int(), che restituisce 0.

12. Disegnare grafici (matplotlib)

La libreria di riferimento per i grafici è matplotlib. La funzione plt.bar(x, y) disegna un grafico a barre dati i valori sull'asse delle ascisse e quelli sull'asse delle ordinate.

Attenzione: il primo blocco con grafici scarica numpy e matplotlib nel browser (può richiedere qualche secondo). 
from collections import defaultdict
import numpy as np
import matplotlib.pyplot as plt

def get_sorted_counts(sequence):
    counts = defaultdict(int)
    for x in sequence:
        counts[x] += 1
    return sorted(counts.items(), key=lambda p: p[1], reverse=True)

years = [1941, 1962, 1941, 1964, 1940, 1941, 1961,
         1963, 1963, 1981, 1962, 1979]

counts = get_sorted_counts(years)
x, y = np.array(counts).transpose()

plt.figure(figsize=(6, 3))
plt.bar(x, y, color="#4f8cff")
plt.title("Prime apparizioni — frequenze assolute")
plt.xlabel("Anno")
plt.ylabel("Numero di supereroi")
plt.show()

Il trucco interessante è np.array(...).transpose() che converte una lista di coppie [(anno, conteggio), ...] in due array separati di ascisse e ordinate. La sintassi x, y = ... sfrutta l'unpacking di Python.

Unpacking. Puoi assegnare più variabili in un colpo solo: a, b = (42, 102). È anche il modo più "pythonico" di scambiare due variabili: a, b = b, a.

13. Leggere dati da file (cenni)

Nei progetti reali i dati arrivano da file (CSV, JSON, database…). Il pattern tipico per leggere un CSV con la libreria standard è il seguente: 

import csv

with open('data/heroes.csv', 'r') as heroes_file:
    heroes_reader = csv.reader(heroes_file, delimiter=';', quotechar='"')
    heroes = list(heroes_reader)[1:]   # esclude la riga di intestazione
  • Il blocco with garantisce la chiusura automatica del file (anche in caso di eccezioni).
  • csv.reader sa gestire separatori diversi dalla virgola (qui il punto e virgola) e campi quotati.
  • list(...) trasforma il "reader" in una lista di righe.
Nota. In questa pagina interattiva non possiamo leggere file dal tuo disco (Pyodide gira in una sandbox). Più avanti useremo pandas: pd.read_csv('heroes.csv', sep=';') fa tutto in una sola riga.

List comprehension

Un costrutto utilissimo per estrarre/trasformare dati da una lista è la list comprehension: [f(e) for e in l if g(e)]. 

years = [1941, 1962, None, 1964, None, 1940, 1981, None, 2099]

# Solo gli anni validi (escludo None) — equivalente a un for + if
solo_validi = [y for y in years if y is not None]
print(solo_validi)

# Conversione e filtro insieme
quadrati_pari = [n*n for n in range(10) if n % 2 == 0]
print(quadrati_pari)

# Outlier? Filtriamoli
plausibili = [y for y in solo_validi if y <= 2025]
print(plausibili)

Quale list comprehension produce [1, 4, 9, 16, 25]?

  • [i for i in range(1, 6)]
  • [i*i for i in range(1, 6)]
  • [i**2 for i in range(0, 5)]
  • [i*i for i in range(5)]
Servono i quadrati di 1, 2, 3, 4, 5: range(1, 6) li genera, e i*i (o equivalentemente i**2) li eleva al quadrato. range(5) parte da 0, quindi includerebbe lo 0.

14. Riepilogo & sfida finale

Quale dei seguenti tipi è mutabile?

  • str
  • tuple
  • list
  • int
Solo le liste (e i dizionari, e i set) si possono modificare in place. Le stringhe e le tuple sono immutabili.

Qual è il modo più idiomatico in Python per scambiare due variabili a e b?

  • tmp = a; a = b; b = tmp
  • a, b = b, a
  • swap(a, b)
  • a = a + b; b = a - b; a = a - b
Lo scambio sfrutta il tuple unpacking: a destra Python costruisce la tupla (b, a), e a sinistra la "spacchetta" nelle variabili.
