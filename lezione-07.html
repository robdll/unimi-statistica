L07 · Analisi della varianza (ANOVA)

Hai osservato un attributo numerico (forza, peso, prezzo, reddito…) suddiviso in più gruppi. Sono davvero diversi tra loro? L'ANOVA risponde confrontando la varianza all'interno di ciascun gruppo con la varianza totale.

≈ 30–40 minuti 7 quiz 3 esercizi Statistica inferenziale

1. L'idea: i gruppi sono davvero diversi?

Hai dati \(x_i^g\) dove \(g = 1, \dots, G\) indica il gruppo di appartenenza. Vuoi capire se le medie nei diversi gruppi \(\overline x^1, \overline x^2, \dots, \overline x^G\) sono significativamente diverse oppure se le differenze osservate potrebbero essere semplice rumore.

L'intuizione: se i gruppi sono identici, la dispersione totale dei dati è ben spiegata dalla dispersione interna a ciascun gruppo. Se invece i gruppi sono diversi, una parte importante della dispersione si genera tra le medie dei gruppi.

2. Scomposizione fondamentale: SST = SSW + SSB

Definiamo:

  • media globale: \(\overline x = \frac{1}{n}\sum_g \sum_i x_i^g\);
  • media del gruppo \(g\): \(\overline x^g = \frac{1}{n_g}\sum_i x_i^g\);
  • somma totale degli scarti: \(\mathrm{SS}_T = \sum_g \sum_i (x_i^g - \overline x)^2\);
  • somma within (entro i gruppi): \(\mathrm{SS}_W = \sum_g \sum_i (x_i^g - \overline x^g)^2\);
  • somma between (tra i gruppi): \(\mathrm{SS}_B = \sum_g n_g (\overline x^g - \overline x)^2\).

Si dimostra che vale l'identità fondamentale

\[ \mathrm{SS}_T = \mathrm{SS}_W + \mathrm{SS}_B \]

Le corrispondenti varianze (campionarie) sono:

  • varianza totale: \(s_T^2 = \mathrm{SS}_T / (n-1)\);
  • varianza within: \(s_W^2 = \mathrm{SS}_W / (n - G)\);
  • varianza between: \(s_B^2 = \mathrm{SS}_B / (G - 1)\).
Idea operativa. Se \(s_T^2 \approx s_W^2\) (cioè \(s_B^2\) piccola): le medie dei gruppi sono praticamente uguali, l'ipotesi di differenza si confuta. Se \(s_B^2\) è grande rispetto a \(s_W^2\): le medie dei gruppi differiscono, l'ipotesi si avvalora.

L'identità \(\mathrm{SS}_T = \mathrm{SS}_W + \mathrm{SS}_B\) è:

  • un'approssimazione valida solo per due gruppi.
  • vera solo se le medie dei gruppi sono uguali.
  • un'identità algebrica esatta, vera per qualsiasi numero di gruppi.
  • vera solo per dati standardizzati.

4. Una funzione anova riutilizzabile

Generalizziamo la procedura per accettare un numero arbitrario di gruppi: 

import pandas as pd
import numpy as np

def anova(groups):
    """Restituisce (var_totale, var_within_pesata)."""
    all_ = pd.concat(groups)
    n = len(all_)
    G = len(groups)

    ss_t = ((all_ - all_.mean())**2).sum()
    ss_w = sum(((g - g.mean())**2).sum() for g in groups)
    ss_b = sum(len(g) * (g.mean() - all_.mean())**2 for g in groups)

    assert abs(ss_t - ss_w - ss_b) < 1e-6, "identità violata!"

    var_t = ss_t / (n - 1)
    var_w = ss_w / (n - G)
    # varianza within "rinormalizzata" sullo stesso denominatore di var_t
    return var_t, var_w * (n - G) / (n - 1)


# Test con due gruppi: forza Marvel vs DC
marvel = pd.Series([85, 100, 55, 75, 30, 100, 13, 19, 10, 55])
dc     = pd.Series([26, 100, 100, 85, 26, 80, 10, 14, 14, 9])
print("Marvel vs DC:", anova([marvel, dc]))

# Test con tre gruppi: peso suddiviso per editore
weight_marvel = pd.Series([191.85, 630.32, 72.97, 135.21, 291.45, 58.96, 81.46])
weight_dc     = pd.Series([95.78, 107.62, 74.74, 146.96, 90.74, 176.32])
weight_other  = pd.Series([66.0, 79.43, 81.26])
print("3 editori:    ", anova([weight_marvel, weight_dc, weight_other]))

5. Interpretazione qualitativa

Dato il risultato di anova(groups), ottieni due numeri: var_t e var_w. Confrontali:

  • vicini: la variabilità totale è tutta «interna» ai gruppi ⇒ le medie dei gruppi sono simili (ipotesi di differenza confutata);
  • var_t molto maggiore di var_w: una parte importante della varianza totale è generata dalle differenze tra le medie dei gruppi ⇒ ipotesi di differenza avvalorata.
Test formale. In statistica inferenziale, il rapporto \(F = s_B^2 / s_W^2\) è messo a confronto con la distribuzione di Fisher per calcolare un p-value. Quel passaggio richiede ipotesi (normalità, varianze omogenee…) e va oltre lo scopo di questa lezione. Qui ci limitiamo a un confronto qualitativo.

ANOVA su pesi: var_totale = 12000, var_within = 11900. Cosa concludi?

  • I gruppi hanno medie molto diverse.
  • I gruppi hanno medie sostanzialmente simili (la differenza è dentro i gruppi).
  • I dati sono distribuiti in modo normale.
  • Un gruppo è chiaramente outlier rispetto agli altri.
Se quasi tutta la varianza totale è spiegata dalla varianza within, significa che le medie tra i gruppi sono simili e la varianza between è trascurabile.

6. Riepilogo & sfida finale

  • L'ANOVA scompone la varianza totale in within (entro i gruppi) e between (tra i gruppi).
  • L'identità SS_T = SS_W + SS_B è una proprietà algebrica.
  • Se var_t ≈ var_w: gruppi simili. Se var_t >> var_w: gruppi distinti.
  • La versione "vera" dell'ANOVA usa il rapporto F per calcolare un p-value: noi qui ci fermiamo a una lettura qualitativa.

SSbetween = 0 implica:

  • tutti i dati sono uguali.
  • tutti i gruppi hanno la stessa media.
  • la varianza è 0.
  • i gruppi sono molto diversi.

Se aumento n dentro a un gruppo (più osservazioni), come cambia tipicamente il rapporto SS_T / SS_W?

  • Diminuisce sempre.
  • Aumenta sempre.
  • Dipende dai dati: l'identità vale ancora ma le proporzioni cambiano.
  • Resta uguale.
