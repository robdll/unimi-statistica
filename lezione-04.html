L04 · Indici di dispersione

Due campioni possono avere la stessa media e raccontare storie completamente diverse: uno concentrato attorno a un valore, l'altro «sparso ovunque». Servono indici che misurino questa dispersione: varianza, deviazione standard, quartili, box plot e Q-Q plot.

≈ 40–55 minuti 9 quiz 4 esercizi Statistica descrittiva

1. Perché serve un indice di dispersione

Considera due eserciti di supereroi con la stessa forza media ma formati molto diversi: nel primo tutti hanno indice 50, nel secondo metà hanno 0 e metà 100. Stessa media, dispersione opposta. La media da sola è un'informazione insufficiente: bisogna affiancarla a una misura di quanto i dati si spargono intorno a essa. 

import pandas as pd

a = pd.Series([50, 50, 50, 50, 50])
b = pd.Series([0, 0, 50, 100, 100])

print(f"a: media = {a.mean():.1f}, std = {a.std():.2f}")
print(f"b: media = {b.mean():.1f}, std = {b.std():.2f}")
Convenzione pandas. .var() e .std() calcolano gli indici campionari, cioè con divisione per n-1 (Bessel correction). Per la versione "popolazionale" usa ddof=0.

2. Varianza, std, quantili

Su una Series pandas:

  • .var() → varianza campionaria \(s^2 = \frac{1}{n-1}\sum(x_i - \bar x)^2\);
  • .std() → deviazione standard \(s = \sqrt{s^2}\) (stesse unità di misura dei dati);
  • .quantile(p) → quantile di livello \(p \in [0, 1]\);
  • .min(), .max() → estremi del campione.
CSV = """\
Name;First appearance
Iron Man;1963
Hulk;1962
Spider-Man;1962
Wolverine;1974
Storm;1975
Thor;1962
Black Widow;1964
Captain America;1941
Doctor Strange;1963
Magneto;1963
Batman;1939
Superman;1938
Wonder Woman;1941
Aquaman;1941
Flash;1956
Cyborg;1980
Joker;1940
Catwoman;1940
Robin;1940
Penguin;1941
"""
import pandas as pd, io
year = pd.read_csv(io.StringIO(CSV), sep=';', index_col='Name')['First appearance']

print(f"min       = {year.min()}")
print(f"max       = {year.max()}")
print(f"mediana   = {year.median():.1f}")
print(f"varianza  = {year.var():.2f}")
print(f"std       = {year.std():.2f}")
print(f"q15       = {year.quantile(0.15):.1f}")
print(f"q75       = {year.quantile(0.75):.1f}")
print(f"IQR       = {year.quantile(0.75) - year.quantile(0.25):.1f}")
print(f"range     = {year.max() - year.min()}")

Tra varianza e deviazione standard, quale ha le stesse unità di misura dei dati?

  • Varianza.
  • Deviazione standard.
  • Entrambe.
  • Nessuna delle due.
La varianza è la media dei quadrati degli scarti: ha unità \([\text{unità}]^2\). Prendendo la radice si tornano alle unità originali — ecco perché la std è spesso preferita quando si vuole "raccontare" la dispersione.

Per ottenere il terzo quartile in pandas, quale chiamata è corretta?

  • s.quantile(3)
  • s.quantile(0.75)
  • s.q3()
  • s.quartile(3)
quantile accetta valori in [0, 1]. Il terzo quartile coincide con il 75-esimo percentile = quantile 0.75.

4. Box plot

Un box plot (o box-and-whiskers) condensa graficamente gli indici principali:

  • la scatola ha basi su Q1 e Q3 (l'altezza è quindi l'IQR);
  • il segmento centrale indica la mediana;
  • i baffi si estendono fino al min e al max — oppure fino a Q1 − 1.5·IQR e Q3 + 1.5·IQR, segnalando come «outlier» i punti oltre.

# Default: gli outlier sono evidenziati con cerchi
w.plot.box()
plt.title("Box plot del peso (con outlier)")
plt.show()

# whis=(0, 100) → baffi fino a min e max, niente outlier
w.plot.box(whis=(0, 100), vert=False, widths=.4)
plt.title("Box plot orizzontale, baffi fino a min/max")
plt.show()

In un box plot di default, un cerchio oltre il baffo destro indica:

  • Il valore della mediana.
  • Il terzo quartile.
  • Un potenziale outlier (oltre Q3 + 1.5·IQR).
  • Un dato mancante.
Con i whiskers di default (1.5·IQR), i punti oltre tale soglia vengono mostrati come singoli cerchi, segnalandoli come potenziali anomalie.

5. Diagrammi Q-Q

Un Q-Q plot (quantile-quantile) confronta due distribuzioni mettendo a coordinate (quantile_di_A(p), quantile_di_B(p)) per molti valori di p ∈ [0, 1]. Se i due campioni hanno la stessa distribuzione, i punti si allineano sulla bisettrice del primo e del terzo quadrante. 

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Due campioni "fittizi" con distribuzioni simili
rng = np.random.default_rng(7)
a = pd.Series(rng.normal(180, 8, 200), name='Marvel')
b = pd.Series(rng.normal(180, 8, 200), name='DC')

levels = np.linspace(0, 1, 100)
plt.plot(a.quantile(levels).values, b.quantile(levels).values, 'o', alpha=.6)

# Bisettrice di riferimento
mn = min(a.min(), b.min())
mx = max(a.max(), b.max())
plt.plot([mn, mx], [mn, mx], '--', color='gray')

plt.xlabel("Quantili A (Marvel)")
plt.ylabel("Quantili B (DC)")
plt.title("Q-Q plot: distribuzioni simili → punti vicini alla bisettrice")
plt.show()

Se le distribuzioni sono diverse, la nuvola si scosta dalla bisettrice in modo sistematico (curve a S, divergenza nelle code, ecc.). 

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(11)
female = pd.Series(rng.normal(63, 6, 150))   # peso femminile
male   = pd.Series(rng.normal(78, 9, 150))   # peso maschile

levels = np.linspace(0, 1, 100)
plt.plot(female.quantile(levels).values, male.quantile(levels).values, 'o', alpha=.6)
plt.plot([40, 100], [40, 100], '--', color='gray')
plt.xlabel("Quantili (F)")
plt.ylabel("Quantili (M)")
plt.title("Distribuzioni diverse: nuvola lontana dalla bisettrice")
plt.show()
statsmodels ha la funzione sm.qqplot_2samples(a, b, line='45') che produce lo stesso risultato in una riga. Per non rallentare la prima esecuzione la teniamo opzionale: se vuoi provarla, attiva data-statsmodels="true" sul blocco di codice.

import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(0)

plt.ylabel("Quantili Esponenziale (scale=5)")
plt.title("Q-Q plot: stessa media, distribuzioni molto diverse")
plt.show()

6. Simmetria, normalità e regola empirica

Una distribuzione è approssimativamente simmetrica quando istogramma e box plot mostrano una forma «a specchio»: nel box plot, baffi di lunghezza simile e mediana al centro della scatola.

L'asimmetria si manifesta come coda:

  • Skew destro (positivo): coda lunga a destra, mediana < media (peso, BMI…);
  • Skew sinistro (negativo): coda lunga a sinistra, mediana > media (es. punteggi su soglia).
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)

simmetrica = pd.Series(rng.normal(24, 2, 1000))
sk_dx      = pd.Series(rng.gamma(2, size=1000))
sk_sx      = pd.Series(rng.beta(20, 2, size=1000))

fig, axs = plt.subplots(3, 2, figsize=(8, 7),
                        gridspec_kw={'width_ratios': [3, 2]})
for ax_h, ax_b, s, t in zip(axs[:, 0], axs[:, 1],
                            [simmetrica, sk_dx, sk_sx],
                            ["≈ simmetrica", "skew destro", "skew sinistro"]):
    s.plot.hist(ax=ax_h, bins=20)
    ax_h.set_title(t)
    s.plot.box(ax=ax_b, vert=False, whis=(0, 100), widths=.4)
plt.tight_layout()
plt.show()

Regola empirica (68-95-99.7)

Quando la distribuzione è approssimativamente normale (a campana e simmetrica), valgono queste percentuali approssimative dei dati entro k deviazioni standard dalla media:

  • ~68% entro 1 std;
  • ~95% entro 2 std;
  • ~99.7% entro 3 std.
import pandas as pd
import numpy as np

rng = np.random.default_rng(0)
sample = pd.Series(rng.normal(180, 7, 5000), name='Height')

m, s = sample.mean(), sample.std()
for k in (1, 2, 3):
    pct = ((sample - m).abs() < k * s).mean()
    print(f"entro {k}σ → {pct:.1%}")

Per la regola empirica, circa quale percentuale dei dati cade entro 2 deviazioni standard dalla media?

  • ~50%
  • ~68%
  • ~95%
  • ~99.7%

Una distribuzione del peso ha una lunga coda a destra. Cosa ti aspetti?

  • media < mediana
  • media > mediana
  • media = mediana
  • Dipende dall'IQR.
La coda destra "tira" la media verso l'alto, mentre la mediana, robusta agli outlier, resta più al centro. Quindi mean > median.

7. Riepilogo & sfida finale

  • Varianza/std: misurano la dispersione attorno alla media; .var()/.std() usano n-1.
  • Quartili e IQR: misurano la dispersione "robusta"; q.quantile(p) per ogni livello.
  • describe(): tutte le statistiche descrittive in un colpo solo.
  • Box plot: scatola = IQR, segmento = mediana; whis=(0,100) per disabilitare il flagging degli outlier.
  • Q-Q plot: stesso pattern lungo la bisettrice ⇒ stessa distribuzione; deviazioni sistematiche ⇒ distribuzioni diverse.
  • Regola empirica: 68-95-99.7 vale (approssimativamente) solo per distribuzioni a campana.

Quale di queste affermazioni è vera per due dataset con la stessa media?

  • Hanno per forza la stessa varianza.
  • Hanno la stessa mediana.
  • Possono avere dispersioni completamente diverse.
  • Hanno lo stesso IQR.

Hai un istogramma con coda lunga a destra. Quale strumento NON è adatto a riassumerlo con un valore "centrale"?

  • Mediana.
  • Media (perché trascinata dalla coda).
  • Quantile 0.5.
  • Moda.
Outlier nella coda spostano la media e la rendono poco rappresentativa. Mediana, quantile 0.5 e moda sono molto più robuste.
